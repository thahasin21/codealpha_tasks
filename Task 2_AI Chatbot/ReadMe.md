💻 Programming FAQ Chatbot
Description

This is a simple FAQ Chatbot focused on programming-related questions. It uses NLP techniques such as TF-IDF Vectorization and Cosine Similarity to match user queries with relevant answers from a predefined dataset.

Features

* Answers programming-related FAQs
* NLP-based question matching
* Streamlit interactive UI
* Fast response generation

Technologies Used

* Python
* Streamlit
* Scikit-learn

How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
